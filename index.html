Currency Converter Application

This dynamic currency converter web application allows users to effortlessly convert amounts between various currencies, leveraging real-time exchange rates.

Key Functionalities:

  • Automatic Exchange Rates: Automatically fetches and updates currency exchange rates for the Euro (EUR) upon page load, ensuring the most current information is used for conversions.
  • User-Friendly Interface: Provides easy-to-use selection options for choosing the source and destination currencies, along with a field for entering the amount to be converted.

This converter exemplifies the integration of external APIs into a web application, showcasing advanced techniques in handling JSON data, managing user events, and creating dynamic user interfaces. It is an ideal tool for websites looking to offer valuable currency conversion services to their users.

A simple project in vanilla JavaScript, themed around a furniture store.

The products are sourced from a JSON file, and for each product, a card is created featuring the photo, price, and an "add to cart" button.
The project started with an HTML-CSS/Bootstrap template without any JavaScript code. Subsequently, each functionality was manually added:

  1. Loading Shopping Cart Data: Calls getCart() function on page load to load shopping cart data.
  2. Displaying Products: Retrieves product data and displays it on the page. Products are dynamically filtered and displayed based on user interactions, such as company selection or price range.
  3. Filter by Company: Allows users to filter displayed products based on the company.
  4. Filter by Price: Users can adjust a slider to filter products based on price.
  5. Search: Includes a search bar to find products based on their title.
  6. Shopping Cart Manipulation: Users can add products to the cart, view a list of added products, modify the quantity of each product directly from the cart, or remove products from the cart.
  7. Displaying Total Price: Calculates and displays the total price of all products in the cart.
  8. Dynamic User Interface: The code updates the user interface based on actions performed, such as adding products to the cart or applying filters, providing immediate visual feedback (e.g., updating the number of items in the cart).
